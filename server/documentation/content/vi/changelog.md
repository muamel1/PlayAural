# Nhật ký thay đổi

## Phiên bản 1.0.0

- Phát hành chính thức PlayAural
- 20 trò chơi: Bài, xúc xắc và phiêu lưu
- Hỗ trợ đầy đủ tiếng Anh và tiếng Việt
- Ứng dụng web (PWA) và ứng dụng Windows
- Hệ thống xếp hạng OpenSkill với bảng xếp hạng
- Hệ thống bạn bè với trạng thái trực tuyến
- Hệ thống trò chuyện toàn cục, bàn chơi và riêng tư
- Giới hạn tốc độ trò chuyện với kiểm duyệt tự động
- Công cụ quản trị và kiểm duyệt
- Hệ thống thông báo máy chủ cho khởi động lại theo lịch
